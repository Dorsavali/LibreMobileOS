About
The LibreMobileOS Foundation, or LMO, is a charitable foundation. Its mission is to support the collaborative development of Secure and customizable mobile operating system. It serves as an independent legal entity to which anyone can contribute code, funding and other resources, with the knowledge that these contributions will be used for public benefit.

Our service
Store
We utilize F-Droid, an open-source app store, as the core of our application distribution. It provides access to a curated selection of free and open-source apps, ensuring transparency, security, and user trust.
Text-to-Speech
For text-to-speech capabilities, we use PicoTTS, an open-source solution designed for lightweight and efficient voice synthesis. This ensures seamless functionality while staying true to our open-source principles.
Cloud Services
Our cloud platform is built on Nextcloud, a self-hosted, secure solution for file storage, sharing, and collaboration. Users can enjoy complete privacy and control over their data via our cloud platform.
Network Location Provider
We leverage MozillaNLP as the foundation for our Network Location Provider service. It ensures accurate location services without relying on proprietary systems, reinforcing our commitment to user privacy. Learn more at our NLP platform.
History
In late 2020, two visionaries, Erfan Abdi and Mohammad Hasan Keramat Jahromi, came together with a shared goal: to revolutionize open-source mobile operating systems. With passion and innovation at its core, the LibreMobileOS Foundation was born.

Infrastructures
At LibreMobileOS, our robust infrastructure is the backbone of our development process, ensuring efficiency, collaboration, and reliability at every step.
  • Gerrit for Code Review:

    We rely on Gerrit as our code review platform to maintain high-quality standards. This powerful tool fosters collaboration among contributors, enabling seamless feedback, tracking, and improvement of code before it becomes part of the project.

  • Git for Code Management:

    Our codebase is securely stored and version-controlled using Git. This ensures that every contribution is meticulously tracked, making it easy to collaborate across a global developer community while maintaining a clear history of changes.

  • Jenkins as Our Build Server:

    Jenkins powers our continuous integration and delivery pipelines. It automates the build and testing processes, allowing us to deliver stable and reliable software efficiently. From compiling code to deploying new versions, Jenkins ensures a smooth and consistent workflow.
